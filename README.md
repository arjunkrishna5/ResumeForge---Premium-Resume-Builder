# ResumeForge

A free, AI-powered resume and CV builder.

## Features
- Guided 7-step resume builder
- Real-time live preview
- AI writing assistance (Gemini)
- Cover letter generator
- 4 professional templates
- Import existing resume (PDF/DOCX)
- Export as PDF or DOCX
- Firebase authentication
- Auto-save to Firestore

## Tech Stack
React 19 · TypeScript · Vite · Express · Tailwind CSS v4
Firebase Auth + Firestore · Google Gemini AI
html2canvas · jsPDF · docx

## Local Setup

1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` to `.env.local`
4. Add your Firebase and Gemini credentials to `.env.local`
5. Run `npm run dev`

See `FIREBASE_SETUP.md` for Firebase configuration.
See `DEPLOYMENT.md` for deployment instructions.
